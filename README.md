# Biostatistics 
Welcome to my biostats repository featuring all of the work I did in 2024 at the University of Dallas.
Most of the data analysis was done in Excell, but I found good moments to use python and R for more fun.

Please do me the favor of checking out my final project, labeled under final_project_west_nile.
It was a really fun challenge trying to write code that I could easily present and explain to an audience of biology majors that didn't know anything about programming while also presenting data driven conclusions over the dataset from California regarding West Nile virus cases.
